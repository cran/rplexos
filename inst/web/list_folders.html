Get list of folders in the working directory

Usage

list_folders()

Get list of folders in the working directory

Description

List of existing folders in the working directory. This function is used when the wildcard symbol ("*") is provided to the process_folder and plexos_open functions.

See also

setwd, process_folder, plexos_open
